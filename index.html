HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML Stands for hypertext markup language.
  • Always remember to use an opening tag as well as a closing tag.
  • The charset element sets our character encoding.
  • The viewport element sets the default view of the webpage.
  • An element embeded within another is known as a child element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.
  • You can use either the color name or it's HTML # to signify which color you want.
  • Remember to link your CSS file sheet to your HTMML doc to ensure changes are activated.

GIT

  • git status: checks which branch we are currently on within the repo.
  • git checkout -b branch name: creates a new branch and swithes to it.
  • You have to initialize your local git repository for it to sync to your GitHub account.
  • GitFlow helps you and other developers keep progress synced.
  • Create a pull request on GitHub to let others review your code before merging to the main branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
