HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML Stands for hypertext markup language.
  • Always remember to use an opening tag as well as a closing tag.
  • The charset element sets our character encoding.
  • The viewport element sets the default view of the webpage.
  • An element embeded within another is known as a child element.

  • GitFlow helps you and other developers keep progress synced.
  • Create a pull request on GitHub to let others review your code before merging to the main branch.
